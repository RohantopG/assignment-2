# 🚀 Generative AI API Integration (Python)

## 📌 Overview

This project demonstrates how to integrate multiple Generative AI APIs using Python.
Each program accepts user input, sends it to different AI providers, and displays the generated response in the terminal.

---

## 🔗 APIs Used

* **Groq**
* **Hugging Face**
* **Google Gemini**
* **Cohere**

---

## ✨ Features

* Accepts input from the user
* Sends requests to multiple AI APIs
* Displays AI-generated responses in the terminal
* Handles errors gracefully
* Uses environment variables for secure API key management

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone <your-repo-link>
cd genai-api-integration
```

2. Install required packages:

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables Setup

Create a `.env` file in the root directory and add your API keys:

```
GROQ_API_KEY=your_groq_api_key
HUGGINGFACE_API_KEY=your_huggingface_api_key
GEMINI_API_KEY=your_gemini_api_key
COHERE_API_KEY=your_cohere_api_key
```

---

## ▶️ How to Run

Run any API script individually:

```bash
python groq_api.py
python huggingface_api.py
python gemini_api.py
python cohere_api.py
```
